# 📊 Student Score Prediction
This project utilizes machine learning techniques to predict student scores based on input features. The objective is to build a robust model that can assist educators and institutions in identifying performance trends and offering targeted interventions.

# 🛠️ Features
- Preprocessing of dataset for clean and efficient analysis
- Exploratory data analysis (EDA) with visual insights
- Model building using regression algorithms
- Performance evaluation with metrics like RMSE and R²
- Interactive prediction functionality for custom inputs

📈 Tech Stack
- Python (Pandas, NumPy, Matplotlib,Seaborn, Scikit-learn)
- Jupyter Notebook
- Multiple Machine Learning Algorithms
