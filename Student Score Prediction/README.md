# 📊 Student Score Prediction
## Project Overview:
This project utilizes machine learning techniques to predict student scores based on input features. The objective is to build a robust model that can assist educators and institutions in identifying performance trends and offering targeted interventions.

## 🛠️ Features:
- Preprocessing of dataset for clean and efficient analysis
- Exploratory data analysis (EDA) with visual insights
- Model building using regression algorithms
- Performance evaluation with metrics like RMSE and R²
- Interactive prediction functionality for custom inputs

## 📈 Tech Stack:
- Python (Pandas, NumPy, Matplotlib,Seaborn, Scikit-learn)
- Jupyter Notebook
- Multiple Machine Learning Algorithms

## Methodology:

1. **Data Preprocessing**: Handling missing values, encoding categorical variables, and feature scaling.
2. **Exploratory Data Analysis (EDA)**: Visualizing the relationships between features and scores.
3. **Model Selection**: Experimenting with various regression models such as Linear Regression, Random Forest, and K Nearest Neighbors.
4. **Model Evaluation**: Assessing model performance using metrics like MSE,RMSE, MAE, and R² score.
5. **Prediction**: Using the best-performing model to predict student scores on new/unseen data.

## Usage:

1. Clone the repository and navigate to the project directory.
2. Install required libraries (e.g., `pandas`, `numpy`, `scikit-learn`, `matplotlib`, `seaborn`).
3. Run the provided Jupyter notebooks or scripts to:
   - Load and preprocess the data
   - Train and evaluate models
   - Make predictions

## Results:

- The best model achieved an R² score of **100%** on the test set.
